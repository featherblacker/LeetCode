###解题思路
两个两个的查, 相异就输出, 注意最后一位

###代码
```python
class Solution(object):
    def singleNumber(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        nums.sort()
        for i in range(0, len(nums)-1, 2):
            if nums[i]!= nums[i+1]:
                return nums[i]
        return nums[-1]
```
